---

##  Project Overview

Pneumonia is a serious lung infection that can be detected through chest X-ray imaging. This project builds a deep learning model to classify X-ray images into:

-  NORMAL
- PNEUMONIA

The model is trained using CNN architecture and optimized using transfer learning techniques for improved accuracy.

##  Dataset Information

- Source: Kaggle
- Dataset Name: Labeled Chest Xray Images
- Categories:

  - PNEUMONIA
- Image Type: Chest X-ray (JPEG format)
